内部の計算¶

  • buoyantPimpleFoamによるエネルギーの非定常計算
  • 閉鎖系で温度境界条件はexternalWallHeatFluxTemperature
    • ①壁の向こう側の温度②熱伝導率③熱伝達率から、境界の温度と勾配が計算される
      • ①-20℃
      • ②0.15mmのAl、上部は(0.15mmのAl+10mmのair)と仮定
      • ③外気の計算から得られた平均値
  • 物性値：組成を(5wt% EtOH + 95wt% H2O)と仮定
    • 物性値は混合比に比例するものとする
      • EtOH : https://www.celsius-process.com/_es/pdf/ethanol.pdf
      • H2O : https://www.celsius-process.com/_es/pdf/eau.pdf

import matplotlib.pyplot as plt
from scipy.optimize import curve_fit

def func(x, a,b,c,d):
    return a*x**3 + b*x**2 + c*x**1 + d

def getPolynomial(dfs_, target, name, EtOH=0.0, show=False):
    para = []
    for specie in ['EtOH', 'water']:
        x = dfs_[specie].index
        y = dfs_[specie][target]
        params, cov = curve_fit(func, x, y)
        para.append(np.array(params))
        if show:
            x_ = np.arange(253,303)
            y_ = func(x_, *params)
            plt.plot(x_, y_, label=specie)
            plt.scatter(x, y, label=specie)
    para = para[0]*EtOH+para[1]*(1-EtOH)
    if show:
        y_ = func(x_, *para)
        plt.plot(x_, y_, label='mean');plt.legend();plt.title(target);plt.xlim(253,303)
    ans = ['{}Coeffs<8>    ( '.format(name)] + ['0']*8 + [');']
    for i, p in enumerate(reversed(para)):
        ans[i+1] = '{:e} '.format(p)
    return ' '.join(ans)

fig = plt.figure(figsize=(15,5))
for k, keys in enumerate([['ρ(kg/m3)', 'rho'], ['Cp(J/kg/K)','Cp'], ['μ(Pas)','mu'], ['k(W/m/K)','kappa']]):
    plt.subplot(1,4,k+1)
    print(getPolynomial(dfs, keys[0], keys[1], EtOH=0.05, show=True))
